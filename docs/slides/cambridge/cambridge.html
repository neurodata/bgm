--- # Summary ![bg center blur:3px opacity:15%](./../../images/background.svg) - Modified the graph matching problem to allow us to incorporate information from contralateral connections in the brain - Proposed an algorithm to solve the problem, which amounts to a simple modification of previous graph matching tools (FAQ) - Showed in simulation that with sufficient correlation in contralateral edges, this can help with matching - Demonstrated that this indeed improves matching in five different bilateral connectome datasets
